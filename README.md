# Sustainable Agriculture Advisor — Plant Health Monitor

---

## Overview

This project implements an end-to-end plant disease detection system for the Sustainable Agriculture domain. A user uploads a photograph of a plant leaf through a web interface; the system classifies the disease across 38 possible classes from the PlantVillage dataset and provides structured agricultural treatment recommendations generated by a language model advisory agent.

---

## System Architecture

```
Leaf Image (Upload)
        |
[Preprocessing: Resize 224x224, Normalize]
        |
[MobileNetV2 / ViT-Tiny Classifier]  -->  38-Class Prediction + Confidence Score
        |
[OOD Check: Confidence >= 15%?]
        |
[LLM Advisory Agent (Gemini API + Plant Disease Ontology)]
        |
Treatment Report (Streamlit Web Interface)
```

---

## Models and Results

| Model | Architecture | Accuracy | Precision (W) | Recall (W) | F1-Score (W) |
|---|---|---|---|---|---|
| MobileNetV2 | CNN — Depth-wise Separable Conv | 91% | 92% | 91% | 91% |
| ViT-Tiny | Vision Transformer (Patch-16) | 34% | 46% | 34% | 30% |

Both models were trained using transfer learning with pre-trained ImageNet weights on a balanced 38-class subset of the PlantVillage dataset (250 images per class).

## Setup Instructions

### 1. Clone the repository
```bash
git clone <repository-url>
cd "Computer Vision PART B"
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Download and prepare the dataset

Download the PlantVillage dataset from Kaggle and place it at `data/PlantVillage/`. Then run the consolidation script to standardize the 38 class labels:

```bash
python consolidate_data.py
```

### 4. Configure environment variables

Copy `.env.example` to `.env` and add your Gemini API key:

```
GEMINI_API_KEY=""
```

The system operates without the API key — it will fall back to the local disease ontology.

### 5. Train the models

```bash
# Full training (recommended on GPU)
python src/train.py --model mobilenet --epochs 25

# Quick local run with a data subset
python src/train.py --model mobilenet --epochs 10 --subset 250
```

### 6. Evaluate on the test set

```bash
python src/evaluate.py --model mobilenet --weights outputs/models/mobilenet_best.pth
```

### 7. Run the web application

```bash
streamlit run app.py
```

The application will be available at `http://localhost:8501`.

## Dataset

**PlantVillage Dataset** (Hughes and Salathe, 2015)

- Approximately 54,000 RGB leaf images
- 38 disease and health classes across 14 plant species
- Source: https://www.kaggle.com/datasets/emmarex/plantdisease
- License: Open-source, academic use

**Data Partitioning:** Stratified 70% Training / 15% Validation / 15% Test split
